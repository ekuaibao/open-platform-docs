获取业务对象

获取某个自建应用下的所有业务对象。

GET/api/openapi/v2/datalink/entity/$platformId

Path Parameters​

名称类型描述是否必填默认值备注
platformIdString自建应用ID必填-自建应用ID获取

Query Parameters​

名称类型描述是否必填默认值备注
accessTokenString认证token必填-通过 获取授权 获取 accessToken

CURL​

curl --location --request GET 'https://app.ekuaibao.com/api/openapi/v2/datalink/entity/$PgQbrAHda85Y00?accessToken=cxEbrzNJSA3A00'

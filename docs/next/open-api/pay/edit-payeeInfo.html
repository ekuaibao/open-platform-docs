更新收款账户

根据收款账户的 ID 修改收款信息。

PUT/api/openapi/v2.1/payeeInfos/$id
更新日志

1.20.0 -> 🐞 接口支持传入 customFields（自定义字段），且响应信息中新增了该字段。
1.14.0 -> 🐞 新增了 nationCode（银行所在地区代码）参数描述。
1.8.0 -> 🐞 修复了更改员工可见性时，偶现的传入启用的员工ID，返回停用的同名员工ID的BUG。
1.6.3 -> 🚀 接口升级 v2.1 版本，新增了 sort（账号类别）参数，额外支持更新 支付宝、海外账号、微信、其他 4种类型收款账户。

Path Parameters​

名称类型描述是否必填默认值备注
idString账户ID必填-通过 获取收款账户 获取

Query Parameters​

名称类型描述是否必填默认值备注
accessTokenString认证token必填-通过 获取授权 获取 accessToken

Body Parameters​

名称类型描述是否必填默认值备注
sortString账号类别非必填BANKBANK : 银行卡
ALIPAY : 支付宝
OVERSEABANK : 海外账号
WEIXIN : 微信
OTHER : 其他
typeString账户类型必填-PUBLIC : 对公账户   PERSONAL : 个人账户
ownerString所有者类型必填-INDIVIDUAL : 个人   CORPORATION : 企业
可以修改所有者类型，不允许修改所有者ID
staffIdString所有者ID非必填-当 owner = INDIVIDUAL 时 必填
当 owner = CORPORATION 时 非必填
可以通过 获取员工列表 获取
nameString开户名称必填-开户名称
cardNoString账号必填-银行卡号

账号
bankString银行名称
『银行卡』专属参数		非必填-可通过 branch 或 bankLinkNo 自动回填，
需保证 branch 在 开户网点 内
branchString开户网点
『银行卡』专属参数		必填-sort = BANK 时，必填。获取 开户网点
可通过系统配置改为 ”非必填“，见下方TIP
bankLinkNoString银联号
『银行卡』专属参数		非必填-当开户网点与合思系统不匹配时，
可通过银联号匹配
provinceString银行所在省
『银行卡』专属参数		非必填-可通过 branch 或 bankLinkNo 自动回填，
需保证 branch 在 开户网点 内
cityString银行所在城市
『银行卡』专属参数		非必填-可通过 branch 或 bankLinkNo 自动回填，
需保证 branch 在 开户网点 内
bankNameString银行名称
『海外账号』专属参数		非必填-银行名称(Bank Name)
swiftCodeString银行国际代码
『海外账号』专属参数		必填-银行国际代码(Swift Code)
nationCodeString银行所在地区代码
『海外账号』专属参数		必填-银行所在地区代码(Nation Code)
传参为 国家地区代码 的数字代码
receiverAddressString收款人地址
『海外账号』专属参数		非必填-收款人地址(Receiver Address)

certificateTypeString证件类型非必填-详细信息见下方【证件类型对照表】
certificateNoString证件号码非必填-证件号码
remarkString备注信息非必填-备注信息
customFieldsObject自定义字段非必填-目前仅支持文本类型（type = text）的字段
可通过 获取全局字段列表 获取
visibilityObject可见范围非必填-可见范围对象，本参数不传，默认为全员不可见
  ∟ fullVisibleBoolean是否全员可见非必填falsetrue : 全员可见   false : 部分可见
部分可见则仅有白名单中可见
  ∟ rolesArray角色白名单非必填-通过 查询角色组和角色 获取
  ∟ staffsArray员工白名单非必填-通过 获取员工列表 获取
  ∟ departmentsArray部门白名单非必填-通过 获取部门列表 获取
  ∟ departmentsIncludeChildrenBoolean下属子部门是否可见非必填truetrue : 可见   false : 不可见
tip

  • 当 branch（开户网点）不确定且必填时，可填写 ”1“（branch不可为null），并保证 bankLinkNo（银联号）正确，系统会根据银联号自动回填开户网点。

  • 当不需要线上支付时，可通过系统配置，关闭 “开户网点” 必填。

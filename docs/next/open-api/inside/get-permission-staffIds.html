获取权限配置员工列表

GET/api/openapi/v1/staffs/getPermissionStaffIds
更新日志

1.25.0 -> 🆕 新增了本接口。

Query Parameters​

名称类型描述是否必填默认值备注
accessTokenString认证token必填-通过 获取授权 获取 accessToken
permissionString权限配置枚举值必填-不支持获取拥有【单据权限】的员工列表

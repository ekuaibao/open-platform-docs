获取业务对象实例列表(条件查询)

根据业务对象实例的 自定义字段 查询数据。

POST/api/openapi/v2/extension/searchAnslysis/entityId

Path Parameters​

名称类型描述是否必填默认值备注
entityIdString业务对象ID必填-通过 获取业务对象 获取

Query Parameters​

名称类型描述是否必填默认值备注
accessTokenString认证token必填-通过 获取授权 获取 accessToken

Body Parameters​

名称类型描述是否必填默认值备注
filedsArray自定义字段数组非必填-自定义字段数组
  ∟ keyString自定义字段名称非必填-可通过 获取业务对象 响应中 items.fields.name 获取
目前支持字段值为 文本、数字、日期 类型字段使用
例如：id、ownerId、createTime
不支持值为 数组 类型字段
多个参数取交集查询
  ∟ valueString 或
Number		查询参数非必填-文本、数字类型传 String
日期（转化为毫秒级时间戳）传 Number
  ∟ operatorString过滤条件非必填=>、>=、<、<=、=、<>
indexNumber开始索引必填-从 1 开始，不可为 0
countNumber查询数必填-每页查询数据量，最大不能超过 100

CURL​

curl --location --request POST 'https://app.ekuaibao.com/api/openapi/v2/extension/searchAnslysis/fa0e76842feec54b3400?accessToken=ID_3Dqicr_0TSw:bwa3wajigF0WH0' \
--header 'Content-Type: application/json' \
--data-raw '{
    "fileds":[     //自定义字段
        {
            "key":"E_fa0e76842feec54b3400_code",
            "value":"B2204-12001",
            "operator":"="
        }
    ],
    "index":1,
    "count":10
}'

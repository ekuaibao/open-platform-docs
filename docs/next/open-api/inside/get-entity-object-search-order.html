获取业务对象实例列表(排序)

根据字段排序业务对象后返回列表数据。

POST/api/openapi/v2/datalink/search/$entityId

Path Parameters​

名称类型描述是否必填默认值备注
entityIdString业务对象ID必填-通过 获取业务对象 获取

Query Parameters​

名称类型描述是否必填默认值备注
accessTokenString认证token必填-通过 获取授权 获取 accessToken

Body Parameters​

名称类型描述是否必填默认值备注
queryObject参数对象必填-包含过滤和排序参数的对象
  ∟ filterByString查询过滤参数必填-过滤参数
  ∟ orderByArray排序参数必填-排序参数
    ∟ valueString排序字段必填=格式：form.E_业务对象ID_code
    ∟ orderString排序规则必填=ASC：升序   DESC：降序

CURL​

curl --location 'https://app.ekuaibao.com/api/openapi/v2/datalink/search/$bd0f8774d3b2f55117c0?accessToken=ID01oMxsLeIf0a%3ADgM3w5DRQ401Iw' \
--header 'Content-Type: application/json' \
--data '{
    "query": {
        "filterBy": "(active.in(true))",
        "orderBy": [
            {
                "value": "form.E_bd0f8774d3b2f55117c0_code",
                "order": "DESC"
            }
        ]
    }
}'

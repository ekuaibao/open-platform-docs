处理EAI插件拉取不到单据中间态问题（待支付、支付中）

POST/api/openapi/v1/docs/setDataBase

Query Parameters​

名称类型描述是否必填默认值备注
accessTokenString认证token必填-通过授权接口获取
powerCodeString功能授权码，传入
219902或者219904即可		必填true219902 : 开放接口   219904 : 开放接口(新)
stateString要修复的单据状态必填-PAYING : 待支付
PROCESSING : 支付中
typeString单据类型必填-expense : 报销单
loan : 借款单
repayment : 还款记录
payment : 付款单
requisition : 申请单
custom : 通用审批单
receipt : 收款单
codesString单据编号必填-单据CODE，多个用英文逗号 , 分隔

CURL​

curl --location --request POST 'https://app.ekuaibao.com/api/openapi/v1/docs/setDataBase?accessToken=ID_3zE5G_0dZw0:bwa3wajigF0WH0&powerCode=219904&state=PAYING&type=expense&codes=B22000002'

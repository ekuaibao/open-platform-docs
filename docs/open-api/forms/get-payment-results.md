### 提交支付结果

{% httpverb "post" %} /api/openapi/v1/paymentResults {% endhttpverb %}


#### Query Parameters:

| 名称       | 类型    | 描述                                                |
| :--------- | :------ | :-------------------------------------------------- |
| **accessToken** | String  |通过授权接口获取。      |


#### Body Parameters示例:
```json
{
	"items":[
		{
			"id":"id1",//单据id
			"code":"P00001",//支付成功
			"errorMsg":null
		},
		{
			"id":"id2",//单据id
			"code":"P00002",//支付失败
			"errorMsg":null
		}
	]
}

```
| 代码      | 含义    | 
| :--------- | :------ | 
| **P00001** | 支付成功  |
| **P00002** | 支付失败  |

errorMsg:出错信息


#### Response返回数据:
| 名称       | 类型    | 描述                                                |
| :--------- | :------ | :-------------------------------------------------- |
| **id** | String  |单据id      |
| **code** | String  |处理结果      |

| 代码      | 含义    | 
| :--------- | :------ | 
| **R00000** | 成功  |
| **R00001** | 单据不存在  |
| **R00002** | 单据已支付  |
| **R00003** | 单据不在支付中  |
| **R00004** | 单据支付渠道不正确  |
| **R00005** | 支付结果代码错误  |
| **R00006** | 无效的请求  |

errorMsg:保留

```json
{
	"items":[
		{
			"id":"id1",//单据id
			"code":"R00000",//更新成功
			"errorMsg":null
		},
		{
			"id":"id2",//单据id
			"code":"R00001",//更新失败
			"errorMsg":null
		}
	]
}
```

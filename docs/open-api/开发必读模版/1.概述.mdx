#### 概念:
<!-- 主要描述该模块的产品定位，一些名词解释，该功能在易快报中的入口，使用步骤等。使客户开发前对此模块有一定基础了解。 -->

数据互联是将异构系统的数据按照约定的格式输入到易快报系统的过程。

数据互联由互联平台构成。每个互联平台对应了一个外部系统，例如「销售易CRM」、「致远OA」等。

每个互联平台可以对应一个或多个互联数据实体。数据实体对应的是外部系统中的业务实体。例如「销售易CRM」中的「客户」、「商机」、「线索」。

通过数据实体的定义，用户可在易快报系统中产生若干份数据实例，例如「销售易CRM」中的「客户」实体，有「客户1」、「客户2」、「客户3」三个实例。



### 应用场景:
<!-- 主要描述该模块适用的核心场景，告知客户在何种业务下需要使用此功能。 -->
数据互联主要满足三个核心场景：

1. 参考外部数据产生易快报系统单据或单据明细。例如：导入订单成为消费明细。
2. 关联外部系统的审批单据。例如：在报销单里关联OA合同审批单。
3. 关联外部系统的基础档案。例如：在报销单里关联CRM系统的客户。

#### 问题一：

Q: 如何获取平台id或实体id？

A: 互联平台id可在易快报【系统设置】-【自定义扩展】页面查看（需管理员权限）。

![平台id](/img/datalink/平台id.png)

点击具体某一平台的配置，进入平台详情页可查看实体id（需管理员权限）。
![实体id](/img/datalink/实体id.png)

***
#### 问题二：

Q: 如何获取一个实体数据的详情？

A: 目前暂不支持通过实体id获取其具体详情。

新增员工/部门扩展字段

POST/api/openapi/v2/property/addPropertySet

Query Parameters​

名称类型描述是否必填默认值备注
accessTokenString认证token必填-通过 获取授权 获取 accessToken

Body Parameters​

名称类型描述是否必填默认值备注
useFieldString字段所属类型必填-STAFF：员工
DEPARTMENT：部门
propertiesArray字段信息必填-字段信息
  ∟ cnLabelString字段显示名称必填-不可传 ""，不可传重复的值
  ∟ enLabelString字段英文名称非必填-字段英文名称
  ∟ canAsDimensionBoolean是否自定义档案非必填falsetrue : 是，档案类型字段必填为 true
false : 否，其他类型字段为 false
  ∟ dataTypeObject数据类型必填-数据类型
   ∟ typeString字段类别必填-所支持类别：
text：文本    number：数字    money：金额
date：日期    ref：引用类型（自定义档案）
   ∟ entityString引用对象非必填-type = ref 时必填，可参考"CURL示例"
格式为：basedata.Dimension. + 档案类别名称
   ∟ unitString单位非必填-type = number 时选填，可参考"CURL示例"
如：kg、cm 等
   ∟ scaleNumber精度非必填-type = number 时必填，可参考"CURL示例"
传入 0 即精度为整数，最多支持8位小数
tip
  • 数据类型有多种情况，可参考"CURL示例"。

CURL​

curl --location --request POST 'https://app.ekuaibao.com/api/openapi/v2/property/addPropertySet?accessToken=ID01n2I9j5xWjR:PCx3rwm3aA00qM' \
--header 'content-type: application/json' \
--header 'Accept: application/json' \
--data-raw '{
    "useField": "STAFF",          //字段所属类型
    "properties":[
        {
            "cnLabel":"档案项目测试01",                 //字段显示名称
            "enLabel": "dims01",                      //字段英文名称
            "canAsDimension":true,                    //是否自定义档案
            "dataType":{                              //数据类型
                "type":"ref",                         //字段类别
                "entity":"basedata.Dimension.法人实体" //自定义档案 
            }
        },
        {
            "cnLabel":"文本01",
            "enLabel": "text01",
            "canAsDimension":false,
            "dataType":{
                "type":"text"  //文本
            }
        },
        {
            "cnLabel":"数字01",
            "enLabel": "number01",
            "canAsDimension":false,
            "dataType":{
                "type":"number",//数字
                "unit":"kg",    //单位
                "scale": 2      //小数位数
            }
        },
        {
            "cnLabel":"金额01",
            "enLabel": "money01",
            "canAsDimension":false,
            "dataType":{
                "type":"money" //金额
            }
        },
        {
            "cnLabel":"日期01",
            "enLabel": "date_01",
            "canAsDimension":false,
            "dataType":{
                "type":"date" //日期
            }
        }
    ]
}'

成功响应​

{
    "items": [
        "档案项目测试01",
        "文本01",
        "数字01",
        "金额01",
        "日期01"
    ]
}

失败响应​

HTTP状态码错误码描述排查建议
400-名称为［文本03］的字段已存在请确认［文本03］的字段是否已存在
400-[number1]为不支持的数据类型请确认 type（字段类别）参数是否在所支持类别中
或所对应的其他必填参数是否填写或填写错误

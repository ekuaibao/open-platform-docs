# 单据的使用规范流程

### 概念:

单据是易快报系统中业务的发起者或触发器。

#### 对于单据有两个视角：

- 配置视角：操作的是单据模板。即企业管理员配置单据模板、设置单据行为。
- 应用视角：操作的是单据实例。即员工编辑单据、审批人审批单据、出纳支付单据。

#### 单据实例通过单据模板生成。单据模板的产生过程为：

- 易快报将不同单据能力与组件组成基础模板。
- 用户（企业管理员）以基础模板为原型，制作单据模板

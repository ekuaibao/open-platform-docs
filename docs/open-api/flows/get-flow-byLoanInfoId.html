根据借款包ID获取借款包信息

GET/api/openapi/v1/loan/loanInfo/$id
更新日志

1.7.1 -> 🆕 新增了本接口。

Path Parameters​

名称类型描述是否必填默认值备注
idString借款包ID必填-可以通过出站消息获取

Query Parameters​

名称类型描述是否必填默认值备注
accessTokenString通过授权接口获取必填-通过 获取授权 获取 accessToken

CURL​

curl --location --request GET 'https://app.ekuaibao.com/api/openapi/v1/loan/loanInfo/$ID_3HV7_Hq1yKw?accessToken=ID_3JxzF8U0aA0:bwa3wajigF0WH0'

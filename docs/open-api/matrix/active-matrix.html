停启用审批矩阵

PUT/api/openapi/v2/matrix/active/$id
更新日志

1.27.0 -> 🆕 新增了本接口。

Path Parameters​

名称类型描述是否必填默认值备注
idString审批矩阵ID必填-通过 获取企业下所有审批矩阵 获取

Query Parameters​

名称类型描述是否必填默认值备注
activeBoolean停用或启用标识必填-true：启用   false：停用
accessTokenString认证token必填-通过 获取授权 获取 accessToken

CURL​

curl --location --request PUT 'https://app.ekuaibao.com/api/openapi/v2/matrix/active/$ID01wh4O1Yl1Cv?active=true&accessToken=ID01wiXBgNwwbR%3APCx3rwm3aA00qM'
